# Documentation

โฟลเดอร์เก็บเอกสารทั้งหมดของโปรเจค

```
docs/
├── report/    # รายงานฉบับสมบูรณ์ (สำหรับ NSC)
└── manual/    # คู่มือติดตั้ง + คู่มือใช้งาน
```

1. **รายงานฉบับสมบูรณ์** (`report/final_report.pdf`)
2. **คู่มือการติดตั้ง** (`manual/installation_guide.pdf`)
3. **คู่มือการใช้งาน** (`manual/user_guide.pdf`)
